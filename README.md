# Loja Social 50+10 – Android App

**Loja Social 50+10** is an Android app to modernize the management of IPCA’s _Loja Social_. It helps the Social Services (SAS) team manage beneficiaries, inventory, and stock efficiently.

## Features

### For SAS

- Manage beneficiaries and inventory
- Schedule deliveries and reminders
- Automatic stock updates
- Expiration alerts

### For the Community

- View available stock
- Information on donations and campaigns
- News and announcements

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Basiiii/LojaSocial-IPCA.git
```

2. Open the project in **Android Studio**.
3. Run the app on an Android device or emulator.
